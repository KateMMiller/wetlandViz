This map displays the results of an ongoing permanent freshwater wetland monitoring program conducted by the National Park Service's Inventory and Monitoring Program. Each circle indicates the location of a permanent wetland monitoring plot. Clicking on a circle will produce a popup with the name of the site and information about the value that is being mapped. The photopoints from that site will also be displayed at the bottom of the window.

  • Sentinel Sites are targeted sites that are monitored intensively following the U.S. Environmental Protection Agency (EPA) National Wetland Condition Assessment methods.
  • RAM Sites are rapid assessment monitoring sites that are randomly located and sampled following the EPA USA-RAM, a manual for Level 1 (GIS-based) and Level 2 (rapid field assessment) methods.

The data and photopoints presented in the Map Panel are from the recent survey of available data. For RAM sites, the data are from 2017-2020, the second sample of each site. For Sentinel sites, the data remain from the initial sample in 2011. As soon as EPA publishes the 2016 vegetation data, we will update this visualizer. Note that protected species have been removed from the data displayed here. Protected species data areavailable upon request and approval by NETN and ACAD staff to distribute.

Map Controls

Type of data: This control allows you to toggle between the different types of data to view. Changing the data type will automatically update the values on the map legend, the data that can be mapped, and the options available for mapping.

  • Veg MMI: is the Vegetation Multimetric Indicator, an index from 0 to 100 that measures and rates the condition of the vegetation at that site.
  • By Site Type: indicates the location of sentinel and RAM sites.
  • By Species: allows you to view species locations on the map.
    • All species allows you to select an individual species to plot locations on the map
    • Invasive species allows you to view all plots where at least one invasive species was detected

Zoom to a Site: Allows you to zoom to a particular site. The selected site will flash light blue after the map zooms in. To go back to full map view, preset Reset Map button.

Reset Map: Resets map to default view and map controls.

Download Data: Downloads the type of data that is displayed in the map for all sites as a .csv.
